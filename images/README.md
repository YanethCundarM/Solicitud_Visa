# Imágenes de Referencia para Pasaporte

## Instrucciones

Por favor, guarda las siguientes imágenes en esta carpeta con estos nombres exactos:

### 1. **pasaporte_portada.jpg**
   - Esta es la imagen que muestra la **portada del pasaporte colombiano**
   - Debe mostrar claramente el **Número de Libreta** (en el recuadro rojo)
   - Ejemplo: AU 281662

### 2. **pasaporte_datos.jpg**
   - Esta es la imagen que muestra la **página de datos del pasaporte** (donde está la foto)
   - Debe mostrar claramente el **Número de Pasaporte** (en el recuadro rojo)
   - Ejemplo: AU271561

## ¿Por qué esto es importante?

Muchos usuarios no saben dónde encontrar:
- El **Número de Pasaporte** (está en la página de datos)
- El **Número de Libreta** (está en la portada)

Estas imágenes de referencia les ayudan a identificar rápidamente dónde está cada información.

## Formato recomendado

- Formato: JPG o PNG
- Tamaño: No mayor a 500x600 px para cargar rápido
- Calidad: Suficiente para ver claramente los números

## Pasos para agregar las imágenes

1. Copia las imágenes del pasaporte colombiano
2. Nómbralas exactamente como se indica arriba
3. Colócalas en esta carpeta (`images/`)
4. Recarga el formulario en tu navegador

¡Listo! Las imágenes aparecerán automáticamente en la Sección 5.
